# PLUGIN DESIGN SYSTEM & VIBE GUIDE

## Core Vibe: "Trae-Inspired AI Modernism"
Mục tiêu là tạo ra giao diện giống như trình soạn thảo Trae: Chuyên nghiệp, tối giản và mang hơi hướng tương lai.

## Visual Rules (UI):
- **Theme:** Dark mode luôn là mặc định.
- **Colors:** - Background: #181818
  - Surface/Card: #1E1E1E
  - Primary Accent: #8B5CF6 (Vivid Purple)
  - Text Primary: #FFFFFF (87% opacity)
  - Text Secondary: #A0A0A0
- **Borders:** 1px solid #333333.
- **Radius:** 8px cho Button và Container.
- **Typography:** Font 'Inter', ưu tiên 'Medium' cho Label và 'Regular' cho Content.

## UX Principles:
- Không có các chi tiết thừa. 
- Hiệu ứng Hover phải mượt mà và tinh tế (ví dụ: đổi nhẹ màu nền).
- Trạng thái Loading phải trông như đang xử lý thông minh (AI processing).
